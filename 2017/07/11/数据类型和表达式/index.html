数据类型和表达式

#scala

val定义:val :=(不可变值)

* 不需要有type指定,编译器可推导

var定义:var := (可变值)

* 不要把var转化类型到不可转化,比如x=1,x=“hello”,数值型可以转.

请优先使用val.

数值类型:

  1. Byte 1字节
  2. Short 2字节
  3. Int 4字节
  4. Long 8字节
  5. Float 4字节
  6. Double 8字节
    上面的是按照等级低到高排序,等级低类型可以转换到等级高类型,而不允许高到低(不会截取),以免丢失数据.但你可以使用toInt()来截取到Int型.
